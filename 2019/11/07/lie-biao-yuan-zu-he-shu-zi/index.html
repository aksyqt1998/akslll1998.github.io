列表元组和数字

1.列表相关操作

1.运算符

1)数学运算符: + , *

列表1 + 列表2 -> 产生一个新的列表，新列表中的元素是两个列表中元素的合并

list1 = [1, 2, 3]

list2 = ['张三', '李四']

print(list1 + list2)

列表 * N / N * 列表 -> 列表中的元素重复N次，产生一个新的列表

print(list1 * 3)

2)比较运算符：# ==，！=，>=，<=(了解)

列表1 == 列表2 -> 判断两个列表的值是否相等

list3 = [1, 2, 3]

list4 = [2, 1, 3]

is

变量1 is 变量2 -> 判断地址是否相等

li1 = [1, 2]

li2 = [1, 2]

print(li1 == li2)

print(li1 is li2) # print(id(li1) == id(il2))

2. in和not in
"""
元素 in 列表       ->  判断列表中是否存在指定元素元素 
not in 列表        ->  判断列表中是否存在指定元素
"""
3.相关函数

len()、max、min、list、sum

1) len(序列) - 获取序列长度（元素的个数）

2) max/min

max(序列)/min(序列) -> 获取序列中元素的最大值/最小值

序列的要求：a.序列中所有的元素的类型一致（数字看成一种类型）

​ b.元素本身支持比较大小

score = [23, 84, 87, 78, 12, 91, 98]

print(max(score))

print(min(score))

3) sum

sum(数字序列) -> 求序列中所有元素的和

scores = [1,2,3]

print(sum(scores))

print(sum(range(101))

4) list

list(数据) -> 将指定的数据转换成列表；数据必须是序列(所有的序列都可以转换成列表)

print(list(100))

print(list('hello'))

print(list(range(10, 15)))

2.列表相关的方法

#####1.列表.count（元素） -> 统计列表中指定元素的个数

names = ['曹操', '刘备', '司马懿', '吕布', '曹操']
print(names.count('曹操'))
2.列表.extend(序列) - 将序列中的元素全部添加的列表中
names.append('后羿')
names.extend('后羿')
print(names)

#####3.列表.index(元素) -> 获取指定元素在列表中的下标

 a.如果元素不存在会报错
 b.如果元素有不多个

#####4.列表.reverse() - 将原来的序列倒序过来（反过来）

nums = [1, 9, 3]
nums.reverse()
print(nums)

nums = [1, 9, 3]
new_nums = nums[::-1]
5.列表.clear() -> 清空列表
nums = [10, 3, 50, 3, 90]
nums.clear()
print()
# 注意:清空列表用clear，而不是直接复制

#####6.复制.copy() - 复制列表中的一个元素，产生一个新的列表

和列表[:]的功能一模一样，都属于浅拷贝
将新列表的地址返回！两个相互不影响

#####7.列表.sot() -> 将列表中的元素从小到大依次排序(直接修改列表元素的顺序,不会产生新的列表)

列表.sot(reverse=True) == 列表.sort();列表.reverse() 从大到小
scores = [89,88,78,79,91,95,93]
scores.sort()
print(scores)

scores.sort(reverse=True)
print(scores)

#####8.排序函数:sorted(序列) -> 不会修改序列，排序后产生一个新的列表

# sorted(序列，reverse=True) -> 不修改原序列，排列后产生一个新的列表(从大到小，降序)
new_scores = sorted(scores)
print(new_scores)

str1 = 'hello'
new_str1 = sorted(str1)
print(new_str1)

# reversed(序列）  ->  将序列中的元素倒序，产生一个新的序列对应的迭代器(了解)
new_str2 = reversed('abc')
str2 = ''
for x in new_str2:    
    str2 += x
print(str2)

3.元组

1.什么是元组(tuple)
"""
元组就是不可变的列表
元组是容器型数据类型，将括号作为容器的标志，里面多个元素用逗号隔开:(元素1，元素2，元素3，....)元组不可变（不支持增删改操作），有序(支持下标操作)
元素可以是任何类型的数据
"""

tuple1 = (1, 2, 3)
print(type(tuple1))

#####2.和列表一样的操作

1)获取元素

列表获取元素的操作元组都支持

tuple2 = (10, 30, 21, 70)

print(tuple2[-1])

print(tuple2[1])

print(tuple2[2:])

2）相关运算: +, *, ==, !=, is, len, max, min, sun ,sorted

print((1, 2, 3) + ( 'abd', 'hello') )

tuple(数据) - 将数据转换成元组(数据必须是元组)

print(tuple('hello'))

#####3.元组特有的或者使用频率偏高的操作

1）单个元素的元组：(元素,)

tuple3 = (10,)

2)单独使用的元组的值，()可以省略 -> 直接多个数据用逗号隔开表示的还是一个元组

tuple4 = 10, 20, 30

3)可以通过让变量的个数和元组中元素的个数来保持一直来分别获取元组中的值

变量1， 变量2， 变量3，…. = 元组

4）同时声明多个变量获取元组中的元素,变量的个数可以比元组中元素的个数少,但是这个时候,多个变量汇中的某一个变量的前面必须加*；带的变量可以获取不带*的变量获取剩下的部分
tuple5 = (10, 20, 299, 19, 230)

4.数字

import math    # 数学模块  -   提供数学相关方法和数据
import cmath   # 提供复数相关数据和方法

python中和数字相关的类型：整型(int)、浮点型(float)、布尔(bool)、复数(complex)

1.整型(int)

所有整数对应的类型(python2.x整型处理int还有long);支持2进制、8进制和16进制的表示方式

int(数据) - 将制定的数据转换成整数；

​ 1）所有的浮点数和布尔都可以转换成整型#

​ 2）字符串，字符串去掉引号后本身是整数

2.浮点型(float)

所有带小数点的数对应的类型;支持科学计数法: 3e4 == 310*4

float(数据) - 将指定的数据转换成浮点数

​ 1）所有的整数和布尔都可以转化成浮点

​ 2）字符串，字符串去掉引号本身就是个数

3.布尔(bool)

只有True和False；True1本质就是1，False本质就是0

bool(数据) - 将指定数据转换成布尔值；

不管什么类型的数据都能转换成布尔，所有为零为空的会转成False，其他会转成True

4.复数(complex)

由实部和虚部组成的数就是复数，实部+虚部j；python直接支持复数的运算

注意：如果虚部是1，这个1不能省

c1 = 10 + 2j

c2 = 5 - 3j

print(c1 * c2)
5.数学模块
print(math.floor(5.2))
print(abs(-12))
print(math.fabs(-20.2))
